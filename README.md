# Ubuntu
> All about Ubuntu/Linux commands

## Minimize on click
### Enable

``` bash
gsettings set org.gnome.shell.extensions.dash-to-dock click-action 'minimize'
```

### Disable

``` bash
gsettings reset org.gnome.shell.extensions.dash-to-dock click-action
```
